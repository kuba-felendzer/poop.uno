Weatherman

A personal utility discord bot

This is a bot that I wrote so wouldn't have to keep sending screenshots of the weather in my personal servers. It uses the python-weather API by vierofernando, which is not the most accurate but it's very easy to use.

Along with telling you the weather, I also added an extra command that just spits out some text it picked from a list, and it can run polls.

This is called Weatherman, and was originally just a weather bot, but nowadays it's just whenever I need/want to add a discord bot feature to a personal server of mine, I add it to this.
